2
3

set tabstop=4

2
3

set shiftwidth=4

禁用烦人的蜂鸣声

  1. 禁用bash蜂鸣声
1
2
3

vim /etc/inputrc 

set bell-style none
  1. 禁用vim蜂鸣声
1
2
3

vim /etc/vimrc 

set noeb vb t_vb=

(vb即visual bell用屏幕的闪烁代替响铃 t_vb就是闪烁设置为空，即不表现出闪烁)
