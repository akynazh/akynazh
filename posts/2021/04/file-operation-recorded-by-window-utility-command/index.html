Window实用命令记录之文件操作

2021-04-19
1 minute read
Code
window , cmd , powershell

创建文件或文件夹

cmd下创建文件

1

type nul > test.jpg

powershell下创建文件

1

new-item test.jpg

创建文件夹

1

mkdir test

重命名文件

注意不能指定到新的路径。

1

PS D:\a\tmp\test> ren test.txt my.txt

移动文件

该命令也可用于重命名。

1

PS D:\a\tmp\test> move test.jpg mydir

复制文件和文件夹

copy无法递归复制，而xcopy可以。

1

PS D:\a\tmp\test\mydir> copy test.jpg test1.jpg

删除文件或文件夹

del

删除文件。powershell可以使用rm。

rd（rmdir）

删除空目录。

rd -R递归删除非空目录。
