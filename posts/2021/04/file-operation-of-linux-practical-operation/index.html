观察文件信息

文件类型+文件权限+目录或链接个数+创建者+所在组+文件大小+修改日期+名称

关于文件类型：

  • d: directory
  • -: file
  • l: link
  • p: pipeline，管道文件
  • b: block，块设备文件
  • c: character，字符设备文件
  • s: socket，套接字文件

[root@VM-0-11-centos test]# mkdir -p d1/d2/d3

# 需先创建目录p，否则复制后会变成 p/d2/d3

[root@VM-0-11-centos test]# mkdir p
[root@VM-0-11-centos test]# cp -R d1 p/
[root@VM-0-11-centos test]# ls
d1  p
[root@VM-0-11-centos test]# cd p/d1/d2/d3
[root@VM-0-11-centos d3]# pwd
/root/test/p/d1/d2/d3

plus:

  1. cp后权限会发生改变，-p保持权限不变

  2. -d：复制时保留链接。这里所说的链接相当于 Windows 系统中的快捷方式

  3. -a: -dpR

文件读写

  • 新建1.txt并进入编辑模式，ctrl+D离开
1

cat > 1.txt
  • 追加写入1.txt
1

echo >> 1.txt
  • 新建test开始编辑并将~/.bashrc作为标准输入
1

cat > test < ~/.bashrc
  • 标准输出和标准错误输出都写入testfile
1
2
3

find /home -name .bashrc > testfile 2>&1 
or 
find /home -name .bashrc &> testfile
