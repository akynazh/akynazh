Dos与Linux换行区别

首先知道：

  • 回车 CR \r
  • 换行 LR \n

win/dos的换行： \r\n

unix mac linux的换行： \n

unix to win 换行失败

win to unix 多了\r -> \r\n -> ^M

文件编码

查看文件编码

enca -L none file

文件编码转为UTF-8

enca -L none -x UTF-8 file

查询历史命令

history 10 查看最近的10笔bash命令记录

!10 执行history中第10笔bash命令记录

修改vim中tab长度

设置vim中tab长度

vim /etc/vimrc

set tabstop=4

禁用烦人的蜂鸣声

禁用bash蜂鸣声

 /etc/inputrc "set bell-style none"

禁用vim蜂鸣声

 /etc/vimrc ":set noeb vb t_vb="

(vb即visual bell用屏幕的闪烁代替响铃 t_vb就是闪烁设置为空，即不表现出闪烁)

永久的alias

为某一个用户永久添加alias

vim ~/etc/.bashrc

alias lm='ls -al | more'

source ~/etc/.bashrc

shell参数

  • $# 参数个数
  • $@ 取出所有参数
  • ${0} 取出第一个参数

变量作用域

name=jzh
echo $name # 输出 jzh
bash
echo $name # 输出 空 
exit
export name
bash
echo $name # 输出 jzh
