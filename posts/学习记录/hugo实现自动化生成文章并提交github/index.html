Hugo实现自动化生成文章并提交github

2021-11-18
2 minute read
学习记录
script , hugo

前言

自动化的思想在计算机领域是非常重要的，计算机，本质上就是为了通过自动化，方便我们的生活，提高办事的效率而存在的。

所以在使用计算机时，在开始用计算机办事之前，可以先想想这件事的那些地方可以利用计算机的自动化来帮我们完成。

正是这种懒的思想，使得人类不断进步，攀登科技高峰，不是吗？^-^

下面介绍通过编写bat脚本自动化hugo写文章，提交发布文章的方法。

Linux用户则使用shell编写脚本~

自动化生成文章

通过 set /p 接收用户输入的文章名称

1
2
3
4
5

D:
cd D:\@git\akynazh
set input=
set /p input=please input article name  
hugo new \post\%input%.md

自动化生成分类

通过copy命令，复制模板文件到categories下并命名

1
2
3
4
5

D:
cd D:\@git\akynazh
set input=
set /p input=please input category name  
copy archetypes\categories.md content\categories\%input%.md

自动提交发布到github

普通提交

方法与以上类似，最后可以pause一下看看有没有问题产生

D:
cd D:\@git\akynazh
hugo
set input=
set /p input=please enter msg  
cd D:\@git\akynazh\public
git add .
git commit -m "%input%"
git push
pause

提交文章

只需要将上面的set部分修改一下即可

1

set input=publish article
