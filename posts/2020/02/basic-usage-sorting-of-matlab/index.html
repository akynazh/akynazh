求解方程组

注：solve已经改版，不可传入字符串

实例：求解二元一次方程

1
2
3
4

syms x y;
s = x - 6 * y == 2;
t = 5 * x + 4 * y == 3;
result = solve(s, t);

解一阶导数方程

1
2
3
4
5

syms x y;
y = -0.0283 * x^2 + 0.7387 * x + 8.9191;
my_x = solve(diff(y)); % diff(A)用于求导数
my_y = subs(y, x, my_x); % 当x=my_x时求y的值
fprintf("(%.2f, %.2f)\n", my_x, my_y);

左除与右除

1
2

右除正斜杠： A / B -> A * inv(B)
左除反斜杠： A \ B -> inv(A) * B

求Ax = b特解

1
2
3

A非方阵时，无法利用x = A^(-1) * b

应该使用x = A \ b
