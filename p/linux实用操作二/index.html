Dos与Linux换行区别

首先知道：

  • 回车 CR \r

  • 换行 LR \n

win/dos的换行： \r\n

unix mac linux的换行： \n

unix to win 换行失败

win to unix 多了\r -> \r\n -> ^M

awk命令操作文本

  • 格式化输出：以tab隔开打印前五行中每行的第一和第三个字符

awk ‘{[pattern] action}’ {filenames}

last -n 5 | awk '{print $1 "\t" $3}'
  • 分隔字符

awk -F #-F指定分割字符

使用",“分隔

awk -F, '{print $1,$2}' log.txt
  • 设置变量

awk -v # 设置变量

设置变量k等于1

awk -vk=1 '{print $1, $1+k}' log.txt

awk -k {awk_script} {file}

使用my.awk处理log.txt

awk -k my.awk log.txt

sed命令操作文本

  • 打印删除了第二到第五行的文本
nl /etc/passwd | sed '2,5d'

  • ‘2,$d’ 匹配删除了第2到最后一行的文本

  • ‘/root/d’ 匹配删除了含root的行的文本

a: add

c: commute

d: delete

i: insert

p: print

s: s/old/new/g

查看内网IP

grep抓取inet行，sed删除inet字串和netmask.*字串

alias myip=" ifconfig eth0 | grep 'inet ' | sed 's/inet//g' | sed 's/netmask.*$//g'"

test:

[root@VM-0-11-centos ~]# myip
         172.17.0.11

查看外网IP

curl cip.cc

show:

IP : ${my_ip} 地址 : 中国 中国

数据二 : 上海市 | 腾讯云

数据三 : 中国北京北京 | 腾讯

URL : http://www.cip.cc/${my_ip}
