Linux实用操作（一）

Dec 03, 2021
阅读时长: 2 分钟

文件编码

查看文件编码

enca -L none file

文件编码转为UTF-8

enca -L none -x UTF-8 file

变量作用域

name=jzh
echo $name # 输出 jzh
bash
echo $name # 输出 空 
exit
export name
bash
echo $name # 输出 jzh

查询历史命令

history 10 查看最近的10笔bash命令记录

!10 执行history中第10笔bash命令记录

禁用烦人的蜂鸣声

禁用bash蜂鸣声

 /etc/inputrc "set bell-style none"

禁用vim蜂鸣声

 /etc/vimrc ":set noeb vb t_vb="

(vb即visual bell用屏幕的闪烁代替响铃 t_vb就是闪烁设置为空，即不表现出闪烁)

永久的alias

为某一个用户永久添加alias

vim ~/etc/.bashrc

alias lm='ls -al | more'

使用cut处理输出的结果

对以:隔开的结果，输出第三个到第五个

echo $PATH | cut -d ':' -f 3,5

使用grep抓取期望的结果

在过去登录主机的人中截取登录信息含root的行信息

last | grep 'root'

比较重要，记忆：

grep [-acinv] [–color=auto] ‘搜寻字串’ filename

选项与参数：

-a ：将 binary 文件以 text 文件的方式搜寻数据

-c ：计算找到 ‘搜寻字串’ 的次数

-i ：忽略大小写的不同，所以大小写视为相同

-n ：顺便输出行号

-v ：反向选择，亦即显示出没有 ‘搜寻字串’ 内容的那一行

–color=auto ：可以将找到的关键字部分加上颜色的显示喔

文件读写

  • 新建1.txt并进入编辑模式，ctrl+D离开
cat > 1.txt
  • 追加写入1.txt
echo >> 1.txt
  • 新建test开始编辑并将~/.bashrc作为标准输入
cat > test < ~/.bashrc
  • 标准输出和标准错误输出都写入testfile
find /home -name .bashrc > testfile 2>&1 
or 
find /home -name .bashrc &> testfile

使用sort和uniq处理结果

使用 last 将帐号列出，仅取出帐号栏，进行排序后仅取出一位

last | cut -d ' ' -f 1 | sort | uniq
