Linux开启ssh服务

  1. 下载OpenSSH

  2. vim /etc/ssh/sshd_config

    编辑配置文件如下：

    1
2
3
4
5

    Port 22 # 开启端口22
PermitRootLogin yes # 允许远程登录
PasswordAuthentication yes # 开启密码验证
ListenAddress 0.0.0.0 # 开启监听
ListenAddress :: # 开启监听

  3. sudo service ssh start开启服务。

    注意ubuntu下为ssh，centos下为sshd。

wsl开机自启服务

  1. wsl下创建脚本，执行：vim /etc/init.wsl，并根据需要写入服务：

    1

    /etc/init.d/${需要的服务}

    再赋予init.wsl执行权限，执行：chmod +x /etc/init.wsl

  2. window下在启动目录下创建脚本：wsl.bat，写入开机命令如下：

    1

    wsl -d Ubuntu-18.04 -u root /etc/init.wsl

    其中，版本通过wsl -l查看。
