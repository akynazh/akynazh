IDEA常用快捷键

alt + ctrl + L : 格式化

ctrl + shift + r : 替换字符

ctrl + r : 在当前文件中替换字符

ctrl + shift + n : 搜索类

ctrl + f : 在当前文件下搜索字符

ctrl + shift + t : 进行快捷抛异常

alt + 上下键 : 光标移到下或上一层
